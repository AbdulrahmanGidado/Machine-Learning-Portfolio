# Machine Learning Portfolio
A collection of end-to-end data science projects focusing on predictive modeling, natural language processing, and business analytics.

## 🚀 Projects Overview

### 1. BigMart Sales Prediction (Regression)
* **Objective:** Predict product sales across various store outlets to optimize inventory.
* **Techniques:** Feature engineering for "Item_Visibility," handling missing values.
* **Models:** XGBoost and Random Forest Regressors.
* **Stack:** Python, Scikit-Learn, Pandas.

### 2. Twitter Sentiment Engine (NLP)
* **Objective:** Classify social media sentiment into three categories (Positive, Negative, Neutral).
* **Techniques:** Custom Regex cleaning pipeline for social media noise, TF-IDF Vectorization.
* **Models:** Logistic Regression.
* **Stack:** Python, NLTK, Scikit-Learn.

### 3. Housing Prices Regression
* **Objective:** Estimate real estate values based on structural and neighborhood features.
* **Techniques:** Linear Regression, Feature Scaling, and Exploratory Data Analysis (EDA).
* **Stack:** Pandas, Matplotlib, Seaborn.

### 4. Titanic Survivor Predictor (Classification)
* **Objective:** Predict passenger survival based on demographic and ticketing data.
* **Techniques:** Data imputation using passenger titles (Mr, Mrs, Miss) to fill missing age values.
* **Models:** Logistic Regression.
* **Stack:** Scikit-Learn, Pandas.

### 5. Superstore Profitability Analysis (EDA)
* **Objective:** Identify "Profit Leakage" and business performance trends in retail data.
* **Techniques:** Advanced data visualization and trend analysis.
* **Stack:** Pandas, Seaborn.
